# to-do-list
Interactive To-Do List web app with clean UI and responsive design.
## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to improve frontend web development skills and practice JavaScript DOM manipulation.

- Add local storage support
- Add task deadlines
- Add dark mode
- Improve UI animations
